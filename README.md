# FKBook Obsidian 插件

一个自定义的 Obsidian 插件。

## 功能

- 基础插件框架
- 设置面板
- Ribbon 图标
- 命令面板集成

## 安装

1. 下载最新 release
2. 解压到 `.obsidian/plugins/fkbook/` 目录
3. 在 Obsidian 设置中启用插件

## 开发

```bash
# 克隆仓库
git clone <repo-url>
cd fkbook

# 安装依赖
npm install

# 构建
npm run build
```

## 发布

使用 GitHub Actions 自动发布，或手动创建 release。
